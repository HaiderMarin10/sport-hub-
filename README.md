# Sport Hub

App web personal de entrenamiento de Andrés Marín — generador de entrenos de
movilidad y fuerza sobre la base de su entrenador (Nacho), repertorio de
ejercicios, generador de WODs estilo CrossFit/Hyrox y temporizador.

Es una **PWA**: se puede instalar en el móvil (Chrome → "Añadir a pantalla de
inicio") y funciona **sin conexión**.

## Publicada con GitHub Pages
Una vez activado Pages, la app vive en:
`https://<usuario>.github.io/<repositorio>/`

## Notas
- Generador, WODs, ejercicios y temporizador funcionan 100% offline.
- Guardar entrenos en Airtable y el Coach IA requieren un backend seguro
  (no incluido en esta versión pública; el token nunca se sube aquí).
